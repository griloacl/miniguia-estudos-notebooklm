# miniguia-estudos-notebooklm
Projeto de estudos da linguagem de programação Python.
**Contexto e Objetivos:** Escolhi python porque em todas as minhas procuras pelo conhecimento de programação é a linguagem com a qual me senti mais a vontade 😀.

**Fontes:** https://docs.python.org/3/ 
            https://www.youtube.com/watch?v=2uBrqwj70TQ
            https://pythonacademy.com.br/blog/7-truques-em-python-que-todo-iniciante-deve-saber

**Prompt:** Qual a melhor estratégia para iniciar nessa linguagem, criando um cronograma para aprimora-la de forma eficaz e leve para não sobrecarregar o dia a dia.

**Saida:** Para iniciar em Python de forma eficaz e sem sobrecarga, a melhor estratégia é focar na prática constante e na construção de uma base lógica sólida, avançando gradualmente dos conceitos básicos para projetos reais
.
Abaixo está uma proposta de cronograma leve, baseada nos fundamentos apresentados nas fontes:
Semana 1: Fundamentos e Ambiente
Foco: Entender o que é a linguagem e como rodar o código.
Atividades:
Utilize sites de execução online (como compiladores de backup) para começar a praticar imediatamente sem a necessidade de instalar ferramentas complexas no início
.
Aprenda sobre Variáveis e os quatro tipos principais: números inteiros (int), decimais (float), textos (string) e booleanos (bool)
.
Pratique a entrada e saída de dados com as funções input() e print()
.
Semana 2: Lógica de Programação e Condicionais
Foco: Aprender como o computador toma decisões.
Atividades:
Domine o uso de condicionais (if, elif, else) para criar caminhos diferentes no seu programa
.
Aplique o Método dos 5Qs para analisar problemas antes de escrever o código: identifique dados de entrada, o que deve ser feito com eles, restrições, resultado esperado e a sequência de passos (pseudocódigo)
.
Semana 3: Repetição e Listas
Foco: Automação de tarefas repetitivas e organização de dados.
Atividades:
Estude laços de repetição: use for quando souber o número de repetições e while para condições que dependem de uma mudança de estado (como um sistema de login)
.
Entenda as Listas, aprendendo a acessar itens por índices (começando sempre do zero) e a adicionar novos elementos com o método .append()
.
Semana 4: "Truques" Pythonistas e Mini-Projetos
Foco: Escrever código de forma mais elegante e resolver problemas reais.
Atividades:
Pratique truques que facilitam o dia a dia, como f-strings para formatar textos, atribuição múltipla de variáveis e reversão de strings
.
Desenvolva projetos pequenos para consolidar o conhecimento, como um calculador de fatorial, um jogo de chute o número ou um medidor de velocidade (radar)
.
Estratégia de Manutenção Eficaz
Consulte a Documentação Oficial: Use o tutorial oficial da versão estável (como a 3.14.6) para tirar dúvidas sobre a sintaxe e funções integradas
.
Prática sobre Teoria: Não tente decorar tudo; a perfeição vem do treino constante
.
Use Materiais de Apoio: Utilize resumos em PDF e códigos prontos para revisar o que foi aprendido sem precisar rever horas de vídeo
.
Aprofunde-se no Guia para Iniciantes: A documentação oficial oferece um "Beginner's Guide" e listas de livros que podem servir de consulta rápida quando você se sentir pronto para o próximo passo
.
